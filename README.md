## About

This repository documents my journey through the **Complete Intro to Web Development v3** on Frontend Masters. It transitions from basic HTML/CSS layouts to complex JavaScript logic and API interactions.

### Project Context

**Purpose**: Master the "Holy Trinity" of web dev (HTML, CSS, JS)  
**Scope**: Full course curriculum including section exercises and final projects  
**Instructor**: [Brian Holt](https://frontendmasters.com/teachers/brian-holt/)

This repository serves as a code laboratory for practicing DOM manipulation, flexbox/grid layouts, and modern ES6+ JavaScript.

---

## Key Learnings

Throughout this course, I implemented:

- **Responsive Design**: Building layouts that work on mobile and desktop.
- **State Management**: Handling user input and calculator memory in JS.
- **Asynchronous JS**: Fetching data from external APIs for real-time validation.
- **Deployment**: Managing a monorepo structure with Vercel root-directory deployments.
